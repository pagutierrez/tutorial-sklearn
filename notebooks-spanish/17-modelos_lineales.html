Modelos lineales¶

Los modelos lineales son tiles cuando disponemos de pocos datos para espacios de caractersticas muy grandes, como en clasificacin de textos. Adems, son una buena forma de estudias los mtodos de regularizacin.

Modelos lineales para regresión¶

Todos los modelos lineales aprenden un vector que contienen los coeficientes, coef_, y una ordenada en el origen, intercept_, que les permite hacer predicciones usando una combinacin lineal de las variables de entrada:

y_pred = x_test[0] * coef_[0] + ... + x_test[n_features-1] * coef_[n_features-1] + intercept_

La diferencia entren los modelos lineales para regresin es que tiene de restricciones o penalizaciones incluyen en el vector coef_ como forma de regularizacin, adems de intentar aproximar bien los datos de entrada. El modelo lineal ms estndar es la regresin de mnimos cuadrados ('ordinary least squares regression'), a menudo conocida como regresin lineal. No pone ninguna restriccin en coef_, por lo que, cuando el nmero de caractersticas es muy alto, no funciona correctamente y puede resultar en sobre-aprendizaje.

from sklearn.model_selection import learning_curve

def plot_learning_curve(est, X, y):
    training_set_size, train_scores, test_scores = learning_curve(est, X, y, train_sizes=np.linspace(.1, 1, 20))
    estimator_name = est.__class__.__name__
    line = plt.plot(training_set_size, train_scores.mean(axis=1), '--', label="puntuaciones de entrenamiento " + estimator_name)
    plt.plot(training_set_size, test_scores.mean(axis=1), '-', label="puntuaciones de test " + estimator_name, c=line[0].get_color())
    plt.xlabel('Training set size')
    plt.legend(loc='best')
    plt.ylim(-0.1, 1.1)

plt.figure()    
plot_learning_curve(LinearRegression(), X, y)

Regresión de cresta (Ridge Regression, penalización L2)¶

El estimador de cresta (Ridge) es una regularizacin simple (llamada regularizacin L2) para el modelo LinearRegression. En particular, tiene el beneficio de no ser ms costoso computacionalmente que la estimacin basada en mnimos cuadrados.

$$ \text{min}_{\mathbf{w},b} \sum_i || \mathbf{w}^\mathsf{T}\mathbf{x}_i + b - y_i||^2 + \alpha ||\mathbf{w}||_2^2$$

La cantidad de regularizacin se ajusta a travs del parmetro alpha del modelo Ridge.

In [7]:

Lasso (penalización L1)¶

El estimador Lasso es til para conseguir imponer dispersin en los coeficientes. En otras palabras, se debera preferir esta penalizacin si creemos que muchas de las caractersticas no son relevantes. Se consigue a travs de la regularizacin L1.

$$ \text{min}_{\mathbf{w}, b} \sum_i \frac{1}{2} || \mathbf{w}^\mathsf{T}\mathbf{x}_i + b - y_i||^2 + \alpha ||\mathbf{w}||_1$$

Modelos lineales para clasificación¶

Todos los modelos lineales para clasificacin aprenden un vector de coeficientes coef_ y un valor de umbral intercept_ para hacer predicciones mediante una combinacin lineal de las caractersticas:

y_pred = x_test[0] * coef_[0] + ... + x_test[n_features-1] * coef_[n_features-1] + intercept_ > 0

Como puedes ver, es muy similar a regresin.

De nuevo, la diferencia de los modelos lineales para clasificacin es el tipo de regularizacin que imponen en coef_ e intercept_, para hay tambin diferencias menores en como se mide la bondad de ajuste con respecto al conjunto de entrenamiento (lo que se conoce tambin como funcin de prdida).

Los dos modelos ms comunes para clasificacin lineal son la SVM lineal, implementada como LinearSVC y la regresin logstica, LogisticRegression.

La influencia de C en LinearSVC¶

El parmetro C de LinearSVC controla la regularizacin que sufre el modelo.

Valores bajos de C llevan a ms regularizacin y a modelos ms simples (permitiendo ms errores), mientras que valores altos de C llevan a menos regularizacin y a modelos ms forzados a intentar clasificar todo correctamente.
