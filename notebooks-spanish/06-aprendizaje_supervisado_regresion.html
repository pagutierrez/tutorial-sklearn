Regresión lineal¶

El primer modelo que vamos a introducir es el llamado regresin lineal simple, que simplemente predice que los datos caen en una lnea recta. Una forma de ajustar este modelo es usar LinearRegression (tambin conocido como regresin Ordinary Least Squares (OLS)). La interfaz para LinearRegression es exactamente la misma que para los clasificadores explorados anteriormente, con la salvedad de que ahora $y$ contiene valores reales, en lugar de valores categricos.

Como recordaremos, el API de scikit-learn requiere que le pasemos la variable objetivo ($y$) como un array de 1 dimensin; el API de scikit-learn espera que los ejemplos ($\mathbf{X}$) vengan en forma de un array de dos dimensiones (aunque solo haya una columna o caracterstica). Por tanto, debemos convertir el array NumPy $x$ de una dimensin a una matriz $\mathbf{X}$ con dos ejes:

In [5]:
print('Antes: ', x.shape)
X = x[:, np.newaxis]
print('Despus: ', X.shape)

Antes:  (100,)
Despus:  (100, 1)

De nuevo dividimos nuestro dataset en entrenamiento (75%) y test (25%):

from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.25, random_state=42)

Ahora usamos el algoritmo LinearRegression para ajustar un modelo de regresin a los datos de entrenamiento:

regressor = LinearRegression(n_jobs=2)
regressor.fit(X_train, y_train)

Despus del ajuste, podemos observar los parmetros obtenidos.

Ya que nuestro modelo de regresin es lineal, la relacin entre la variable objetivo ($y$) y la variable independiente ($x$) se define como:

$$y = peso \times x + \text{ordenadaOrigen}$$

Para representar el modelo, introducimos los valores mximo y mnimo de $x$ en esta ecuacin.

In [9]:
min_pt = X.min() * regressor.coef_[0] + regressor.intercept_
max_pt = X.max() * regressor.coef_[0] + regressor.intercept_

plt.plot([X.min(), X.max()], [min_pt, max_pt])
plt.plot(X_train, y_train, 'o');

De nuevo, scikit-learn proporciona un mtodo fcil para evaluar cuantitativamente los estimadores de regresin a travs de la funcin score. Para tareas de regresin, este sera por defecto el R2. Otra forma popular de evaluacin sera el Mean Squared Error (MSE). Como indica su nombre, el MSE consiste simplemente en la diferencia al cuadrado promediada entre los valores predichos y los valores reales:

$$MSE = \frac{1}{n} \sum_{i=1}^{n} (\text{predicted}_i - \text{true}_i)^2$$

KNeighborsRegression¶

Del mismo modo que para clasificacin, podemos utilizar un mtodo basado en vecinos para regresin. Podemos predecir directamente el valor de la variable objetivo del punto ms cercano o podemos promediar varios de los puntos ms cercanos. Los mtodos basados en vecinos son ms populares en clasificacin que en regresin, pese a lo cual es un mtodo sencillo que puede servir como baseline.

In [15]:
from sklearn.neighbors import KNeighborsRegressor
kneighbor_regression = KNeighborsRegressor(n_neighbors=1)
kneighbor_regression.fit(X_train, y_train)
Out[15]:
KNeighborsRegressor(algorithm='auto', leaf_size=30, metric='minkowski',
          metric_params=None, n_jobs=1, n_neighbors=1, p=2,
          weights='uniform')

De nuevo, vamos a comprobar como se comporta en entrenamiento y en test:

In [16]:
y_pred_train = kneighbor_regression.predict(X_train)

plt.plot(X_train, y_train, 'o', label="datos", markersize=10)
plt.plot(X_train, y_pred_train, 's', label="prediccin", markersize=4)
plt.legend(loc='best');

En entrenamiento, la prediccin es perfecta, ya que cada ejemplo es su vecino ms cercano.

En el conjunto de test, capturamos mejor la variacin de los datos que con el modelo lineal, aunque nuestro estimador haga algunas cosas un poco raras. Evaluemos la puntuacin R2:

Es mucho mejor que antes. El problema era que un modelo lineal no era un ajuste adecuado para el problema, ya que no tena la suficiente complejidad e infra-entrenaba los datos (error alto por sesgo).
